# Binomial Distribution Simulator (Python + NumPy + Tkinter)

## Project Description

This repository contains a Python project that demonstrates the Binomial Distribution using simulation and visualization.

The program includes a graphical user interface built with Tkinter that allows users to input parameters for a binomial experiment such as the number of trials, probability of success, and number of simulations.

The script generates random outcomes using NumPy and visualizes the results using Matplotlib and Seaborn.

This project was created to demonstrate probability distributions, statistical simulations, and data visualization using Python.

---

## Technologies

- Python
- NumPy
- Matplotlib
- Seaborn
- Tkinter

---

## Project Included

### Binomial Distribution Simulator

File: main.py

This project simulates the binomial distribution and visualizes the results using a histogram.

The program includes a graphical interface where the user can enter:

- number of trials
- probability of success
- number of simulations

Features include:

- generating random binomial outcomes
- displaying calculated simulation results
- plotting the distribution of outcomes
- interactive graphical interface using Tkinter

---

## How to Run

Install the required libraries:

pip install numpy  
pip install matplotlib  
pip install seaborn  

Run the program:

python main.py

---

## Example Usage

Example input:

Number of trials: 10  
Probability of success: 0.5  
Number of simulations: 1000  

The program will generate random binomial outcomes and display a histogram showing the distribution of successes.
